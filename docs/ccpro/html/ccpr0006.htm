$# CAMCAD - Professional Edition

· Mentor 4 File Reader (ASCII)

CAMCAD Professional needs the following Mentor Files. The currently supported Mentor system is Boardstation V8.x.

1.Geometry ASCII file with all geometry included. This file has the defintions of drawings, geometries, padstacks and so on.

2.Component file, which is called in the Mentor system comps.comp.$x. This file has the component placement information.

3.Netlist file, which is called in the Mentor system nets.net.

4.Trace or Wire file, which is called traces.trace. This file has all the routing and filled area information.

When loading the Mentor files, the Geometry ASCII file must be loaded first (like on the Mentor system). There after, once this file is loaded, all other files can be loaded in any order or combined.
