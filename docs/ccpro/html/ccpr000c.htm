· PADS PCB Reader

Component shapes and types in PADS.

PADS has a section DECAL and PARTTYPE.

DECAL is the physical, graphic description of a shape (i.e. DIP14).

PARTTYPE is a defintion of components which used one or more decals (alternates). The PARTTYPE is what places the component. A PARTTYPE also can change pinnames of a DECAL.

CAMCAD does the following:

DECAL : All DECALS are converted to Geometries.

PARTTYPES : are attached as a ATT_TYPE to a component.

If a PARTTYPE has alphapins, a new GEOMETRY (DECAL) gets created called TYPE_DECAL and the component gets this Geometry attached.

The CAMCAD Typelist must have a unique identification between component and its defintion. PADS can change a PARTTYPE with alternate Decals.
