· Install & Upgrade Instructions

1) Verify that your access code sheet has CAMCAD PROFESSIONAL printed at the top. If you have an old access code sheet that does not have CAMCAD PROFESSIONAL on the top, please contact Router Solutions for a current access code sheet.

2) If you are installing CAMCAD from diskettes, simply place disk 1 in your floppy drive and launch the setup.exe program. Follow the on-screen instructions to complete the install.

3) If you are installing CAMCAD from a downloaded file, simply find and double-click on the file from your file navigator. When prompted for the password, enter it exactly as it is printed on your download instruction sheet.

4) CAMCAD will install to the "CAMCAD" directory, and will create icons found under Start | Programs | CAMCAD (Win95, 98, NT4.0) or a CAMCAD Group & icons (NT3.51).

5) After you install CAMCAD PROFESSIONAL on your system, you can install new upgrades / versions "on top of" each other, meaning that the same install directory can be selected for each new install or upgrade of CAMCAD. The install will detect your current CAMCAD files, and will only install new or updated files to your system. This will speed up your installation / upgrading process considerably.

6) The icon created in the installation to launch CAMCAD PROFESSIONAL runs the following command line: c:\camcad\camcad.exe /PROFESSIONAL (or c:\[directory]\camcad.exe /PROFESSIONAL.

If you wish to create another icon to launch CAMCAD PROFESSIONAL, you must modify the icon properties and enter the /PROFESSIONAL command flag.
