$# CAMCAD - Professional Edition

· VeriBest PCB Reader

The EIF output processor generates multiple EIF files. The easiest way to import is to use the Correspondance file, but also single loading of files is allowed. CAMCAD needs the

following files as minumum:

PAD

PADSTACK

CELL_LIB

DESIGN

NETLIST

Typcial Correspondance file example:

.FILETYPE CORRESPONDENCE

.VERSION "VB14.00"

.CREATOR "Intergraph"

.PAD "eifo_pad.txt"

.PADSTACK "eifo_pin.txt"

.CELL_LIB "eifo_cel.txt"

.CLEARANCE "eifo_clr.txt"

.TECHNOLOGY "eifo_tec.txt"

.DESIGN "eifo_des.txt"

.PDB_ASCII "asciipdb.txt"

.NETLIST "netlist.kyn"
