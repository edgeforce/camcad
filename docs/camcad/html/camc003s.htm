$#Poly Query

Modify, Attribs, Close buttons and Keyword/Value Window

The last features of the Poly Query dialog box are:

•Modify: Applies any changes made in the Query dialog box to the entity in the project window. You must use this button to insert your changes.

•Attribs: Brings up the CAMCAD attribute editor. For further information about the attribute editor, refer to the specific help section called Attribute Editor.

•Close: If you select Close without clicking Modify, you will loose your changes.

•Keyword/Value window: Shows you the defined attributes and associated values for your selected entity. This is the same information that can be found in the attribute editor, but the editor will allow you to modify the settings. The Keyword/Value window only displays the settings.
