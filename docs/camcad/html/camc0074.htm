$# Deleting Non-Electrical Items

On initial import, the document has all the design layers displayed. Generally, a test engineer is not interested in the trace, unroute, copper pour area, mechanical diagram, miscellaneous documentation, or note layers. These layers fill the database with items not normally important to testing and tend to clutter the image and make redrawing a slow process.

You can delete the non-useful items. However, you do want to preserve the primary board outline.

Do not overwrite the original file.

To permanently delete unused non-electrical documentation items from the CAMCAD database:

1.Remove non-useful data:

a.Select Tools > PCB > Delete Generic Graphic. The Generic Delete dialog box appears.

b.Select Visible, Non-Electrical Graphic, then click OK.

c.The Delete is Not Reversible dialog box appears. If you are sure you want to proceed, click Yes.

2.Select View > Redraw. The board outline is restored.
