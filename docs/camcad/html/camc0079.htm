$# Locating a Specific Part or Position on the PCB

Although this information is available in the standard CAMCAD Help file, it is also described here because the functions are commonly used.

To find a specific component:

1. Select Zoom > Pan to Reference Designator. The Pan to Reference Designator dialog box appears:

The Pan to Reference Designator Dialog

2.Enter the reference designator of the component that you wish to pan to, then click OK.

The specified component is centered in the display and is selected (changed to the highlight color). If the display is set to Zoom Extents, then only the specified component is selected and highlighted.

To pan to a specific location on the PCB:

1. Select Zoom > Pan to Coordinate. The Pan to Coordinate dialog box appears:

The Pan to Coordinate Dialog

2. Enter the desired coordinates into the dialog box, then click OK.

The specified coordinate is centered in the display.

If the display is set to Zoom Extents, no action is taken.
