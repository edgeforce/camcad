$#Add > Layer

Adds a new layer to a file.

In CAMCAD, you have a project window in which multiple files can be displayed, and each file can have multiple layers. After you select Add > Layer, you will be prompted for a new layer name. This new layer is assigned a color which can be changed later.

Adding layers is extremely helpful in editing. You can:

•Select and move any entity from one layer to another.

•Turn layers on and off to display only the information that you need.
