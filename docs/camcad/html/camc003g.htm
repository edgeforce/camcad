$#Graphic Classes Toolbar

Offers shortcuts to several View commands from the menu command bar. Click on the links below for specific command information.

View > Show Fills

View > Show Attributes

View > Show Classes > Show All Classes

View > Show Classes > Hide All Classes

The following buttons are on the Show Toolbar, and they indicate individual classes that may be toggled on or off for display purposes:

nrm = normal

ana = analysis

a_e = anti-etch

b_g = board geometry

drw = drawing

drc = DRC

etc = etch

mnf = manufacturing

rki = route keep in

rko = route keep out

pki = place keep in

pko = place keep out

vki = via keep in

vko = via keep out

aki = all keep in

ako = all keep out

bol = primary board outline

col = primary component outline

pol = primary panel outline

mil = milling

urt = unroute

tfo = trace fanout
