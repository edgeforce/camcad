$# Viewing Component Attributes via Query

Attributes are only encountered in intelligent files. Files such as Gerber, DXF, or HPGL files do not contain embedded attribute information, such as REFDES or VALUE.

To view the attribute data about a given part:

1.Select a component by clicking on it.

2.Select Query > Edit Entity.

3.The Query dialog box appears, displaying any information known about the item.

4.Click Attribs. The Attributes dialog box appears, displaying all attributes assigned to that geometry.

From this dialog, you can edit, delete or add attributes.
