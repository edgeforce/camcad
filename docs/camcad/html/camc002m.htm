$#View > Current Settings

Toggles the display of the Current Settings dialog box.

The Current Settings Dialog Box

The Current Settings dialog box displays the following information:

•Current File ListBox: Displays the filename of the currently selected entity, or the filename to which changes/additions will be made.

•Current Layer ListBox: Displays the layer of the currently selected entity, or the layer on which any changes will be made.

•Current Width ComboBox: Displays the current width (aperture) setting for the highlighted entity, or the width (aperture) that will be used to draw any new entities.

•Scale: Displays the scale for the currently selected entity.

•Rotation: Displays the rotation for the currently selected entity, or the rotation which will be applied to new entities.

•Text: Displays the text size (in inches) for the currently selected text, or the size in which inserted text will be printed.

•Mirror: Toggles whether inserted text will be mirrored.
