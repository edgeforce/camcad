$#Settings > System

You will find two sections under Settings > System:

•Load Menu Language File: Allows you to change the language file used to write the menu bar and menu commands for international support.

•Save English Menu Language File: When you save the English Menu language file, you create a .txt file whose contents consist of the text and formatting of the CAMCAD menu bar. You can then manually edit this .txt file to change the menu commands to the language of your choice.
