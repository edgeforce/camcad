#Text Query

The Text Window And Font Dropdown Choices

The top of the Text Query dialog box contains a window that shows the selected text.

1.Click inside the window to bring up a cursor, and change, modify, add, or delete the text.

2.Click Modify to apply the changes to the project window.

The font dropdown selection will list all CAMCAD .fnt files that are currently loaded into your system. If you have particular True-Type fonts that you wish to use, you can download fontgen.exe from the RSI web page. This program will convert any True-Type font into a CAMCAD readable .fnt file. Place the .fnt file in the CAMCAD root directory, and it will become a choice in the font dropdown choices in the Text Query dialog box.
