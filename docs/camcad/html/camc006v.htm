CAMCAD Read/Import Name R-CADIF

Full CAD Format NameVisula & CADSTAR-NT

Database ClassNet: Connections are allowed to be made in any order and “T” connections are also allowed. Although rules must be followed for trace routing and via placement they are usually very loose, thus allowing the designer much more freedom in the routing process.

Board Outline ClassReal: Board outlines in this CAD format are true board outline entities. CAMCAD supports true board outlines.

Component ClassLibrary: Library driven CAD systems have a component library section within the CAD data which defines all components.

Items controlled by the Cadif.IN file

.COMPOUTLINE

Sets the default layer which is the Primary Component Outline

•Syntax: .COMPOUTLINE layername

•Example: .COMPOUTLINE L241

.MIRRORLAYER

Establishes default layer mirroring

•Syntax: .MIRRORLAYER layer to be mirrored mirrorlayer
