$# Viewing Component Attribute Information via Navigator

Attributes are only encountered in intelligent files. Files such as Gerber, DXF, or HPGL files do not contain embedded attribute information, such as REFDES or VALUE.

The Navigator tool lets you access individual components of the database by moving through the file-type structure.

1.Select Tools > Navigator.

2.The Navigator dialog box appears, displaying the file-type structure of the database. Generally, you will see entries for Files, Geometries, and Layers. You can:

Click the + symbol to expand a portion of the structure.

Click the - symbol to deflate a portion of the structure.

4.Click the + symbol on the Board folder and the + symbol on the Board file. Double-click the Board icon. An Attributes window appears. Click X. A dialog box opens showing device reference designators.

5.Drag the slider bar to find an entry with the name of any device reference designator. In this case, the designator is diode_b “CR704.” Select the entry, then click Edit Attributes.

6.The Attributes dialog box displays all the available information in the database about this device.

From this dialog, you can edit, delete or add attributes.

Editing, Adding, or Deleting Attributes

Any attribute you edit, delete, or add is immediately posted to the database. If you delete data you may have to do another import to get the data back!
