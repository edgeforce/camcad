# Aperture Query

A & B Sizes, X & Y Placement

When an aperture is queried, you can view the defined A and B sizes in the Aperture Query dialog box. If you need to modify the size, click in the appropriate field and type in the new size.

Note

Once a change is made and you click Modify, the change is global for that particular aperture. All instances of the aperture will have the new size settings.

Likewise, the X and Y placement fields will display the current coordinates for the aperture in question. If you wish to move the selected aperture to another location, modify these fields.
