$#Tools > Navigator

Opens the CAMCAD Navigator dialog box. Three folders will be shown initially:

•Files: When expanded, you will see all of the files that are loaded into the current project window. Depending on the files you have loaded, different information will be available. Information such as component instance, netlist, and component type will be displayed with all attributes.

•Geometries: When expanded, all geometries contained in the files that are loaded will be displayed. This includes global pad, drill, padstack, and component definitions along with any defined attributes.

•Layers: When expanded, all layers for the currently selected project window. Defined attributes in the layers can also be seen in this folder.

To view, edit, add, or delete attributes for any desired entity, select that entity and click Edit Attributes.
