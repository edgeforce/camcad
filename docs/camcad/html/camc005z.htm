$# User Interface

The CAMCAD user interface uses standard Microsoft Windows operations. Non-functional or unlicensed feature menu picks are grayed out.

The CAMCAD screen has the following elements:

The top line shows the program title, software release version, and name of the currently open document.

The second line is the main menu.

The third line is a Smart Icon toolbar.

The main area of the screen contains the current document flanked by other Smart Icon floating/docking toolbars to the left, right, and/or bottom.

The bottom line has a status bar containing the current document’s scale, X and Y coordinates of the cursor in the view, and the dimensional units of the coordinate system.
