$# Working With Layers

Many different file formats contain multiple layers of data. Each layer has a separate name, and is represented graphically by different colors. All of the layers contained within all of the files currently loaded into the project window are visible with the Tools > Layer List command.

The Layer List Dialog

Individual layers can be toggled on or off, and specific information regarding each layer can be set. At the left side of the Layer List window, the layer number is on a button. Click on a layer button.

The Layer Edit Dialog

The Layer Edit dialog will appear, offering settings specific to just that layer. For example, you can set the Z Coordinate on the Silkscreen layer for use in extruding component height for export to the Solidworks 3-D modeling system. The top of the Layer List dialog shows the Layer List Mode. Each mode offers further settings and modifications that are possible.

Navigate to View > Toolbars and toggle the Current Settings dialog on. Then, select different entities in your file. Carefully note the information that is displayed in the Current Settings dialog, and how it changes from entity to entity.

The Current Settings Dialog

The Current Settings shows the currently selected file, layer, and width, as well as other selected options. It is important to note that this dialog will:

•Display information about selected entities

•Show what settings will be used for any added entities

For example: You decide to add some text to your data. Issue the Add Text command, and insert the text. The Current Settings dialog will tell you what layer the text is inserted on, what width it was drawn with, etc. This also applies to added lines, polylines, apertures, and all other inserted entities.
