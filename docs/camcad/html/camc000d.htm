$# Menu Commands

Select any of the CAMCAD menu choices below for further information. Entries in italics are not yet implemented.

File commands allow you to import, export, and print.

Zoom commands allow you to alter the view of the layout in the project window.

Edit commands allow you to manipulate the CAD data in the CAMCAD environment.

Query commands allow you to find out further information about any entity in the CAD database, or mark and highlight entities.

Add commands allow you to create new entities in the CAD database.

Tools commands allow you to view and manipulate the CAD data in a variety of ways.

Report commands allows you to generate reports in a comma-delimited file that contain information about the CAD data.

Settings commands allow you to change the CAMCAD environment.

View commands allow you to change the display of toolbars or entities within the CAD data.

Window commands allow you to display two or more windows in a variety of ways.

Help commands allow you to view the help files or enter access codes.
