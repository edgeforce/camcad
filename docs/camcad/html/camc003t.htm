$# Polyline Display Rules

In CAMCAD, all polyline draws will follow the following rules:

•All verticies which are not endpoints will be displayed as round.

•Endpoints of lines and polylines will be displayed as Round, Square, or Endcap.

•For polylines drawn with a Round or Square aperture, all verticies will be displayed as round. For polylines drawn with a Rectangle aperture, all verticies will be displayed as square.

•Square line ends will be handled with a rotated aperture.

•Endcap line ends will be handled with a rotated aperture.

Gerber Output Rules:

•Extended (RS274x) Gerber Output: Extended Gerber output must support rotated apertures

•Standard (RS274) Gerber Output: Must warn users if non-othogonal lines and/or polylines drawn with a Square or Endcap exist. These features will be assigned a new round aperture of same width as the original geometry

Rotated Apertures In General:

Square and round apertures can be handled by CAMCAD. Rectangular apertures cannot be rotated.
