$#Show Insert Types Toolbar

Offers shortcuts to several View commands from the menu command bar.

The following buttons are on the Show Insert Types Toolbar, and they indicate individual insert types that may be toggled on or off for display purposes:

unk = unknown

via = via

pin = pin

pcbcmp = pcb component

mchcmp = mechanical component

gencmp = generic component

pcb = pcb

fdc = fiducial

tool = tooling

tpb = test probe

frpd = free pad
