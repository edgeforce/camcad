$#Query > Query/Edit Entity

Provides specific information on whatever entity is selected.

The Query > Query/Edit Entity dialog box differs depending on the type of entity that is selected, as different entities have different attributes. At the top of the query dialog box is a word that identifies the kind of entity that is being queried, such as Poly, Aperture, Text, and Insert.
