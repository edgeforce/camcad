#Text Query

Height, Width, Angle, And Oblique Settings

Determines how the text in question is displayed on the screen.

•Height: Refers to the height of the text.

•Width: Tells you the width of the field assigned to each letter.

•Angle: Refers to the angle at which the line of text is drawn. For example, an angle setting of 0 means the line is written horizontally, while an angle setting of 90 will cause the text to be written vertically.

•Oblique: Allows you to italicize your text. Positive entries in this field will result in text leaning to the right, and negative entries will cause the text to lean to the left. The useful range of settings is -50 to +50, with 0 being no lean.

Text adjustment is possible in the Text Query dialog
