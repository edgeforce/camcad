$#Write Modules : DXF

File > File Export > DXF Write

See also: File > File Import > DXF Read

DXF Information:

Exports DXF files that are compatable with several different target systems, including Autocad R12, R13, and R14. One notable exception is CADKEY — if you are looking to generate a file for import into CADKEY, you should not use the DXF Write option in CAMCAD. Rather, you should obtain the CADKEY CADL Write module, and use that to generate a CADKEY readable file for import.

When you export your DXF file, you will see the specific export parameters defined in the DXF Export Format dialog box. These options are discussed in further detail below.
