$#Tools > Keyword Map Editor

The Keyword Map Editor lets you to set attribute mapping information, so that proper attribute information is exported.

For example, you may need to map the attribute TOLERANCE. CAD data often uses the attribute names +TOLER and -TOLER, but the HP3 070 requires the name be +TOL and -TOL. CAD data also often uses INSTPAR, but the HP 3070 requires VALUE. Therefore, mapping is required for correct export of the data.

To map keywords:

1.Select Tools > Keyword Map Editor.

2.The Keyword Map dialog box shows the keywords and how they will be used on both input and output.

3.To edit a keyword:

a.Select a CAMCAD keyword (for example, TOLER), then click Edit Keyword. The Edit Keyword Map dialog box appears.

b.Enter the new name in the Export Keyword field (for example, TOL), then click OK.

4.Repeat step 3 to map additional keywords as necessary.

5.Click Done.
