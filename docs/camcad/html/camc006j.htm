$# Viewing and Selecting a Device

When viewing the graphics, any item can be selected (highlighted). Additionally, you may need to locate a specific device or X, Y location.

To: Do This:
Select a given device or item 1. Zoom into a small area of the display.

2.Click on the item.

3.The item changes to the specified select color when it is selected.

Frequently, the item you try to select is adjacent to another item. Within the search range CAMCAD will select the first item in the database within the search range. Pressing F3 will cycle between all items.

If the component is re-selected, it is because no other entities are within the select range. By default, the range is 300 pixels.
Locate a specific device (pan to) 1. Click Zoom > Pan to Reference Designator.

2.Type the device reference designator (for example, R202) and click OK. The view is centered at the designated device location, and the device is highlighted.
Move to a specific X,Y location 1. Click Zoom > Pan to Coordinate.

2.The Pan to Coordinate dialog box appears. Type in the coordinates (for example, 2.307, 3.745) and click OK.

The view is centered at the designated X, Y location.
