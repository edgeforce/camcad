$#File > Print

Prints the contents of the working project window.

When selected, the print dialog box appears with several different options:

Print Options

•Extents Of Drawing: When toggled, prints the entire drawing, whether or not the drawing is visible or only partly visible in the project window.

•Contents Of Window: When toggled, forces the printer to print only what is visible in the current project window.

•B & W / Color: Allows you to toggle between black & white or color printouts.

•Fit To Page: Rescales your image to fill the entire page when printed.

•Print To Scale: Prints the contents of the project window in the scale it is displayed, regardless of the paper size. The "scale" is user-definable in the edit box that is underneath the Print To Scale option.

•Print By Layer: Forces the printer to only print one file layer at a time, each layer on a different page.

•Print Banner: Toggles on and off the "Printed By CAMCAD - [filename]" banner that normally appears at the bottom of all printed pages.

•Paper Orientation - Portrait / Landscape: Allows you to select the way you would like your file to print on a page. Portrait refers to normal, 8.5" X 11.0" printing, and landscape refers to sideways 11.0" X 8.5" printing.
