$# Entering Access Codes

Your access code sheet indicates a user name and company name. Temporary code sheets also have an expiration date and the duration in days. To enter access codes:

1.Launch CAMCAD by double-clicking on the program icon.

2.Select Help > About CAMCAD > Access Codes.

3.Enter the specified user and company name from your Software Certificate. If your code sheet is temporary, enter that information as well. Otherwise, leave the temporary fields blank.

4.Activate each module you purchased:

a.Double-click on a module name.

b.In the Access Code dialog box, enter your access code, then click OK.

c.Full Access appears next to the module name.

d.Repeat this sequence for each modules on your access code sheet.

Installation is complete.

Access Codes are 8 hex characters (0-9; A, B, C, D, E). The codes do not use the letters “O” or “I,” but do use the numbers “0” or “1.” If you have problems entering your access codes, go to Access Code Problems.
