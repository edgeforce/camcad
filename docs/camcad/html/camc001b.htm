$#Edit > Move Selected

Moves objects or entities in a CAMCAD project window.

1.Use the mouse to select the item(s)you want to move.

2.Select Edit > Move Selected.

3.Move the mouse pointer so that the crosshair is near the selected item, and left click. The selected item is "picked up" and is movable with the mouse.

4.When the item is in the new location, left click again. The item is "dropped."

To select multiple objects at once, press and hold down the shift key while selecting each individual object.
