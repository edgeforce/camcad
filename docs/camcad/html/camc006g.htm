$# Basic Steps - Exporting

To export a file:

1.Select File > Export or click .

2.In the Export Format dialog box, choose your file type from the selections on the Licensed side. (If the type you are looking for is on the Unlicensed side, contact RSI to obtain the proper export module access code.)

Depending on the file type you choose for the export, you may have a dialog box that comes up to change export parameters. Usually the defaults are fine to use; however, you may want to explore the "Write/Export Modules" section of this help file to find out the specifics of the export parameters for each export module that we offer.

3.Choose a filename and location for the exported file.

Default file extensions are automatically assigned, such as .asc, .brd, .gbr, or .plt, depending on the export format that you choose.

You do not have to use these extensions, and you may choose whatever file name that works best for you.

More specific export information is available in the CAMCAD help file for your specific version of CAMCAD.
