$# Saving the Keymap File

Once attribute keyword mapping is complete, you can save a map file to retain the attribute mapping.

Future conversions are easier and faster when you can take advantage of map files!

1.Click Save Map.

2.The Save As dialog box appears. Either select an existing file name or enter a name containing the.kwd suffix in the File Name field (for example, cctour.kwd). Click Save.

3.Close the Keyword Map dialog box. The keyword mapping is complete.
