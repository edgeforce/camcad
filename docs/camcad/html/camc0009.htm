$# Help On Help

The CAMCAD help file is organized into the following sections:

Getting Started: This section covers general CAMCAD information and is useful to review if you are unfamiliar with or new to CAMCAD. Topics covered include the user interface; commonly-used toolbars; general module information; CAMCAD Edition differences; file and database basics; installing and starting CAMCAD; single user and network setup information; and uninstallation information. QuickStart guides you through importing sample files.

Importing Data:This section covers the steps you need to perform in order to import data into CAMCAD. Ten import modules are also outlined here.

Manipuatling Data:This section covers ways in which you can manipulate data in CAMCAD. Topics include working with attributes; editing the keyword map; selecting, adding and editing entities;and working with layers, apertures, geometries, and netlists.

Exporting Data:This section covers the steps you need to perform in order to export data into CAMCAD.

Command Reference:This section covers menu commands, toolbars, and CAMCAD hotkeys.

Frequently Asked Questions:This section lists answers to many of the issues you are likely to face.

For More Information:This section gives you contact information for RSI if you need further assistance.

A number of help pages are accessible through the index. Click the Index button to view the topics.

How Commands are Described

When navigating through a series of commands to perform an action, shortened annotation is used. For example, the command sequence for opening the help file - "Find and click on the word Help on the menu bar to drop down a list of choices, and select Index" will be shortened to "select Help > Index." Similarly, importing an HPGL file will be written as"select File > Import > HPGL Read" instead of "On the menu bar click on File, then select Import, and then select HPGL Read.”
