# Aperture Query

The D-Code Dropdown Choices

In the Aperture Query dialog box, the top field refers to the D-Code of the aperture in question.

To change this aperture to any other defined aperture:

1.Click on the dropdown arrow at the right of the D-Code field.

2.Browse through the list of possible selections, and choose the aperture that you need.

Note

When a new aperture is selected, the shape and size fields will be modified to reflect the proper attributes of the new aperture.

3.Once the aperture is found, click Modify to make the change to the project window.
