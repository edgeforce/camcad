$#Poly Query

Layer Dropdown Choices

Shows you what layer the queried entity lies on.

To modify the layer:

1.Drop down the list of choices.

2.Select the layer that you want to move the polyline to.

3.Click Modify to make the change.

When the screen is redrawn, you will see the polyline appear in the color of the new layer, showing you that the polyline was moved to the new layer.
