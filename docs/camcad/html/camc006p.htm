$# Globally Assigning Part Attributes

You can globally assign a part attribute (not to be confused with keyword mappings) using the Navigator. When changing attributes on a global part, you must refresh the database so that all of the same parts inherit the change.

The figures in this section use the “terminal” part with REFDES E200 and E201 in the neutral.cc file.

1.Select Tools > Navigator.

2.In the Navigator dialog box, expand the Geometry folder.

3.Scroll to locate the desired geometry (for example, “terminal”) and select it.

4.Click Edit Attributes. The Attributes dialog box appears.

5.Select Add. The Edit Attributes dialog box appears.

6.In the Keyword field enter TEST, then click OK.

7.The attribute TEST now appears in the Attributes dialog box. Click Done.

To update the global attribute inheritance:

1.Select Tools > PCB > Find/Pan Component. A Pan to Reference Designator dialog box appears.

2.Enter E200, then click OK. The terminal E200 will be colored on the PCB drawing.

3.Click Tools > Refresh Attribute Inheritance.

You must do a Refresh Attribute Inheritance for the new attributes to be applied to all of instances of the geometry.

4.Click Query > Edit Entity. The Insert dialog box appears.

5.Select the E200 and E201 parts and verify that the attribute TEST was inherited and added to the Keyword list.

Changes made to individual parts are immediately inherited.
