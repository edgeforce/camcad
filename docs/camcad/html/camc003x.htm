# Aperture Query

Shape Dropdown Choices

The Aperture Query dialog box lets you view and modify any selected aperture's shape using the shape field.

Note

All changes are global for the current project window. For example, if D-10 is normally round, and you select and query a D-10 aperture and change the shape to square, all instances of the D-10 aperture will be changed to square when you click Modify.
