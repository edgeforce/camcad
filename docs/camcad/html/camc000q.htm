$#File > New

Creates a new project window in CAMCAD.

The default titles of the new windows are CAMCAD1, CAMCAD2, CAMCAD3, etc. until a project is loaded into the new window.

You can import one or more files into a CAMCAD project window. There is no limit on the number of files that you may import into any project window, and there is no limit on the number of project windows that you can have open at the same time.

Note

Computer/system resources must be sufficient to handle the projects.
