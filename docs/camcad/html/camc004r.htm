$# Selecting Entities

Entities may be selected in CAMCAD in a variety of ways. Move the mouse crosshairs over an entity and left click. The center of the crosshairs contains a circle that represents the Search Range.

Tip

Press F3 to scroll through entities contained within the search range until the desired entity is highlighted.

The Search Range diameter is set to a default of 300 pixels. You can modify this diameter by selecting Settings > Modify Settings > System, Search Range field.

•Query > Mark by Window and Query > Mark by Window (Cross-check) allows you to select groups of entities .

•Mark by Window allows you to drag and draw a window. Any entity that is fully contained within the window will be marked.

•Mark by Window (Cross-check) also allows you to drag and draw a window. Any entity partially contained or fully contained within the window will be marked.

•Query > Toggle Mark for Entity unmarks an entity. Clear and reverse marked commands are also found under the Query Menu commands.

•Zoom > Pan to... selects specific entities.
