$# Deleting Non-Useful Traces

When working with files that have non-useful traces, you can permanently delete these layers from the CMCAD database:

1.Select Tools > PCB > Delete Traces, Vias. The Delete Traces, Vias dialog box appears.

2.Select with Netnames from the Traces dialog box, then click OK.

The Delete is Not Reversible dialog box appears. If you are sure you want to proceed, click Yes.
