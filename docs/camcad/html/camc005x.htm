$# Working With Netlists

Netlist information is only relevant when refering to intelligent CAD files. Graphic-only formats such as DXF, Gerber, or HPGL files contain no netlist information. Users of CAMCAD Graphic Edition can skip this section.

1.Import a CAD file and display it on the screen. It is often helpful to be able to highlight and display netlist information.

Note

Turn on the PCB Toolbar for shortcuts to netlist functions.

2.Select Tools > PCB > Select Nets. A dialog containing a listing of all the nets present within the CAD data appears.

3.Choose a net, such as GND, and click Mark and Highlight. The GND traces will be highlighted in the display.

A related useful function is being able to select a trace and then highlight everything electrically connected to that trace (on the same net). Select and highlight a trace and select Tools > PCB > Highlight Selected Item's Net.
