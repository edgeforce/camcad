$#Edit > Delete Polyline Vertex

Removes a vertex from a polyline.

1.Select Edit > Delete Polyline Vertex to enter "edit" mode

2.Move the crosshairs over the polyline and vertex you want to remove.

3.Click the left mouse button, and the vertex on the polyline nearest the crosshairs is removed.

4.Press the [ESC] key to deselect the command and exit "edit" mode.
