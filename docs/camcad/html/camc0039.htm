$#Tools > File List

Displays the File List dialog box, which shows all the files currently loaded into the project window.

1.Click on ON or OFF just to the left of the file name to toggle individual files as on or off.

2.Click on a filename to highlight it, and click Modify.

3.The Edit File Placement dialog box displays. This allows you to manipulate:

•the X and Y insertion point of the file

•the scale of the file

•the rotation of the file

4.Click inside one of the fields to bring up a cursor, then type in the desired value.
