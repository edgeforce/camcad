#Text Query

Proportional, Mirror, And Negative Checkboxes

These three settings further modify the queried text.

•Proportiona: Toggles whether each character of text is given the same amount of room, or whether the room allocated will be proportional to the text character.

•Mirror: Mirrors the text about its insertion point when toggled.

•Negative: Causes the text to be placed in the negative plane - so it is scratched instead of drawn.
