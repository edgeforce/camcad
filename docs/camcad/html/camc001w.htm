$#Add > Line

Add > Line > Line Segment

Inserts a line segment into the project window.

1.Select Add > Line to enter "add" mode.

2.Move the mouse in your project window, clicking once to begin the line, and once to end the line.

3.After you get the line(s) placed, select Add > Line > Line Segment again to exit "add" mode.

Add > Line > Polyline

Inserts a polyline into the project window.

1.Select Add > Polyline and insert your new polyline into the current project.

2.After you have the polyline inserted, select Add > Polyline End or Add > Line > Polyline to exit "add" mode.
