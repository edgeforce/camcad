$#Query > Query/Edit Entity > Poly Query

When you query a polyline, the Poly Query dialog box displays the attributes of the polyline.

The dialog box contains:

•A window that has Poly, Poly - CLOSED, or Poly - CLOSED FILLED, indicating the state of the polyline.

•A listing of the X and Y coordinates of the vertices of the polyline.

•The specific attributes for the polyline.

If you are attempting to query a line or arc, you will probably see the query come up as a poly query.
