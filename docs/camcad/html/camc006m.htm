$# Editing, Adding, or Deleting Attributes

To edit a components attributes:

1.Go to the Attributes dialog box.

(See Viewing Component Attributes via Query or Viewing Component Attribute Information via Navigator.)

2.Select an attribute (for example PARTNUMBER).

3.Click Edit. The Edit Attributes dialog box appears, displaying the value of the item.

4.Enter any changes to the Value or Visible Attributes fields, then click OK.

Any value changes are reflected in the Attributes dialog box.

5.To complete the edit, click Done.

6.To close the Attribute dialog box, click the X box in the upper right corner.
