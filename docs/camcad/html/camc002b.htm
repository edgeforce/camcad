$#Tools > PCB

Tools > PCB > Highlight Net

Brings up a dialog box in which you select a net name to highlight. When you click OK, it highlights the selected net on the board.

Tools > PCB > Highlight Selected Item's Net

Highlights the net of the selected item, if the selected item is part of a net.
