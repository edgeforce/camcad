Defining Undefined Apertures

Working With Apertures

Lets you edit apertures and manipulate the aperture listing.

The Aperture List dialog contains specific, detailed information about all of the defined apertures in the currently selected project window.

•Click on the column header to sort by different criteria.

•Click on selected aperture's shape icon to change the aperture shape.

•Double-click on an aperture's name to edit the aperture.

•Click on selected aperture's name to edit the name of the aperture.

Double-clicking on an aperture name will bring up the Edit Aperture window
