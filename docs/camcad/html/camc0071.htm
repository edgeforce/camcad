Database ClassNet: Connections are allowed to be made in any order and “T” connections are also allowed.  Although rules must be followed for trace routing and via placement they are usually very loose, thus allowing the designer much more freedom in the routing process.

Importing Protel into CAMCAD

File(s) Used For Import

.PCB (ASCII)

How To Produce File(s)

In the Protel for Windows environment perform the following steps:

1. Select Save As from the File Menu.

2. Select the Text option.

3. Name the file.

4. Select OK.

Notes On File Generation

Items controlled by the Pfw.IN file

.BOARDOUTLINE

Sets the default layer for the Primary Board Outline.

.COMPOUTLINE

Sets the default layer to find the Primary Component Outline.

.HEIGHTATTR

Sets what CAMCAD will recognize as the component height attribute.

.MIRRORLAYER

Establishes default layer mirroring.

.ComponentSMDRule

Establishes whether components should be flagged as SMD depending on a series of rules.

.COMMENTATTR

Sets the attribute name for the general comment field present in Protel.
