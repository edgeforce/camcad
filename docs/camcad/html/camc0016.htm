$#Zoom > Pan To Text

Lets you search through your drawing for a text string.

The Zoom > Pan To Text command brings up a dialog box with several user-definable settings:

•Text field: Enter the text string to search.

•Case Sensitive: Toggles whether text case is relevant or not.

•Find Substrings: Use this when the text you are looking for is a substring of a larger text string. For example, D-14ARPOL1 is a text string containing POL1. If you issue the command Zoom > Pan To Text and search for the text string POL1, the search will fail unless Find Substrings is toggled as on.
