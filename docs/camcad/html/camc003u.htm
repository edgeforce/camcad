$#CAMCAD Hotkeys

The following keys are shortcuts to commands in CAMCAD:

CTRL + N............................................ File > New
CTRL + P............................................ File > Print
PGUP............................................ Zoom > Zoom In
PGDOWN.................................................. Zoom > Zoom Out
HOME........................................................ Zoom > Zoom Extents
INS............................................................ Zoom > Pan To Center
F2.............................................................. Zoom > Next Text
CTRL + Z.................................................. Edit > Undo
CTRL + A.................................................. Edit > Redo
CTRL + D.................................................. Edit > Delete Selected
F3.............................................................. Query > Find Next
CTRL + Q.................................................. Query > Query/Edit Entity
CTRL + M................................................. Query > Toggle Mark For Entity
