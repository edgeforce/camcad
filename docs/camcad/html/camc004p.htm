$# Adding New Entities

Use the Current Settings Toolbar to set qualities for the new entity to have (such as what file it should be in and what layer it should be on).

Pick the file from the Current File ListBox that you want to add the entity to. You can add a new file with the Add File command.

Pick the layer from the Current Layer ListBox you want the new entity to be on. You can add a new layer with the Add Layer command. If Locked Layerlist is set in the System Settings, only the layers for the selected file appear in the Layer ListBox in the Current Settings.

For lines, polylines, circles, arcs, and text, pick the appropriate width (aperture) from the Current Width ComboBox to draw the entity with.

•If you type a numeric value in the Current Width ComboBox, a new round aperture will be created with this size and the new entity will be drawn with this aperture.

•If you type a name in the Current Width ComboBox, a new round aperture will be created with this name and a zero size and the new entity will be drawn with this new aperture (for example, "D10").

•If you want to change the size or shape of an aperture, you can edit apertures in the Aperture List

When adding a text entity, the Current Text Height, Current Mirror, and Current Rotation can be modified on the fly before placing the text. The results can be seen while dragging the new text.

When adding an insert of a geometry, the Current Scale, Current Mirror, and Current Rotation can be modified on the fly before placing the geometry. The results can be seen while dragging the new insert of the geometry.

These current settings can be set by hand, or you can highlight another entity with the desired qualities (file, layer, width, etc...) and CAMCAD will fill the current settings appropriately.

Example 1: Adding a new line to an empty window: Add a file, add a layer to the file, then add the line

Example 2: Adding a line to an existing database: Pick the file, layer, and width (aperture) for the new line, or highlight an entity that has the desired file, layer, and width, then add the new line(s).
