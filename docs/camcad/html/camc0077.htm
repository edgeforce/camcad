$# Mirror/Rotate the PCB

Currently, the surface access called out in the board_XY file does not represent the mirrored PCB. This will be rectified in the next release.

When the PCB design database is imported into CAMCAD, it is oriented the same as it was in the original CAD system. This orientation may not be ideal for viewing or fixturing. CAMCAD lets you rotate and mirror the PCB.

To rotate or mirror the PCB:

1.Choose Tools > File List. The File List dialog box appears.

The File List dialog box displays all currently loaded files. In some cases a single CAD database may yield multiple files within CAMCAD.

2.Using the mouse, select the file that represents the PCB design, then click Modify. The Edit File Placement dialog box appears:

The Edit File Placement Dialog

3.Rotate and/or mirror the board:

a.To rotate the PCB, enter a new rotation angle in the Rotation field.

For HP3070 fixturing it is strongly recommended to keep the principle axis of the PCB in alignment with the principle axis of a test fixture (normally in 90 degree increments). Normally, the rotation is applied counter clockwise about the origin of the PCB. To rotate clockwise, use a negative value in the rotation field.

b.To mirror the board, check the Mirror check box.

CAMCAD allows a file to be mirrored in the y-axis only. However, by using a combination of mirroring and rotation, a flip in both the x-axis and y-axis can be achieved.

Tip

If a mirror in both the x and y- axes is desired, simply check the Mirror check box and apply a rotation of 180 degrees.

c.Click OK.

d.In the File List, click either Apply (to keep the File List open) or OK (to apply rotation and close the File List).

Once the new rotation is applied, the PCB image may no longer be visible.

To reset the page size around the new PCB location:

1.Select Settings > Modify Settings. The Setting dialog box appears:

The Modify Settings Dialog

2.Select the Page tab.

3.Select Fit Page to Image, then click OK.

The window redraws with the entire PCB now visible.
