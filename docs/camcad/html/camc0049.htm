#Insert Query

Inserted Geometry Name And Dropdown Choices, Ref Name Field

In the Insert Query dialog box, you will first see the inserted geometry name of the queried entity. To change this entity to another defined entity:

1.Drop down the list of choices in the Inserted Geometry field.

2.Select a new type of geometry.

3.Click Modify and the change is made.

Note: The change will only affect the selected geometry. Other instances of the original geometry type will not be affected.

The Ref Name field is the reference name assigned to the queried geometry. If no reference name is assigned, this field will be blank.
